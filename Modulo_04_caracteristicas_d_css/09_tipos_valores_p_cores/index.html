CSS - Tipos de Valores para Cores

Temos algumas formas de poder estar adicionando cores em nosso site, como:

  • Nome:usa o nome da cor em inglês. Ex.: red, blue, green.
  • Hexadecimal: É uma combinação de números e letras que representam a quantidade de vermelho, verde e azul. Ex.: #ff0000 (vermelho), #0000ff (azul). Muito usado e aceita cores mais precisas.
  • RGB: Significa Red, Green, Blue. Você define a intensidade de cada cor de 0 a 255. Ex.: rgb(255, 0, 0) (vermelho). Permite criar qualquer cor misturando os três valores.
  • RGBA: É igual ao RGB, mas adiciona o Alpha, que controla a transparência de 0 (totalmente transparente) a 1 (opaco). Ex.: rgba(255, 0, 0, 0.5) (vermelho com 50% de transparência).
  • HSL: Significa Hue (matiz), Saturation (saturação), Lightness (luminosidade). Matiz: de 0 a 360 (cobre todas as cores). Saturação: 0% a 100% (sem cor até cor forte). Luminosidade: 0% (preto) a 100% (branco). Ex.: hsl(0, 100%, 50%) (vermelho puro). Mais fácil de criar tons claros ou escuros.
  • HSLA: É o HSL com Alpha para controlar a transparência. Ex.: hsla(0, 100%, 50%, 0.5) (vermelho com transparência).
  • currentColor: currentColor funciona dentro do próprio elemento onde ela está sendo usada. Ela pega o valor da propriedade color daquele elemento e aplica em outras propriedades como borda sombra, fundo, etc.
