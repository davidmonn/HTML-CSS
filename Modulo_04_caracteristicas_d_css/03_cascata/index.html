CSS - Cascata

A ordem de prioridade da cascata (do mais forte para o mais fraco) é:

  • CSS Inline (escrito direto no HTML, com style="...")
  • CSS Interno ( dentro da tag style no HTML )
  • CSS Externo (em arquivos .css separados)

Ou seja: se houver conflito, o estilo inline vence, depois o interno, e por último o externo.
Porém, isso só vale quando a especificidade é a mesma. Se o CSS externo usar um seletor mais específico (como um ID), ele ainda pode vencer o inline.
