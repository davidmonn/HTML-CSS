CSS - Unidade de Medida Relativas (2/2)

As unidades de medidas relativas, iram mudar de tamanho baseado nos elementos que temos em nossa pagina.

Unidades Relativas:

  • em: é baseada no font-size do elemento pai. Se o pai não tivesse um font-size definido, o filho herdaria do body (16px), ou seja: 1.5 * 16px = 24px.

  • rem (Root EM): É uma unidade de medida relativa ao tamanho da fonte definida no elemento raiz (html). Não importa o tamanho da fonte dos elementos pais, o rem sempre usa como base o valor do font-size do html. Por padrão, nos navegadores, o font-size do html é 16px (salvo se for alterado).

  • % (Porcentagem): A unidade % usa como base o valor disponível no elemento pai, não o tamanho da tela. Ela calcula a porcentagem em relação ao tamanho do elemento no qual está inserida, ou seja, é relativa ao elemento pai ou ao contexto em que está aplicada.

  • ex É uma unidade de medida relativa à altura da letra “x” da fonte usada no elemento. Ou seja, o valor de 1ex corresponde à altura do caractere “x” minúsculo na fonte atual. É menos usada porque o tamanho da letra “x” pode variar bastante entre fontes diferentes.
  • ch É uma unidade de medida relativa à largura do caractere “0” (zero) na fonte usada no elemento. Ou seja, 1ch equivale à largura de um “0” na fonte atual. É útil para criar elementos com largura proporcional à quantidade de caracteres (ex: caixas de texto).

Nos exemplos abaixo, temos um box pai com um font-size de 20px.

em:
height: 10em - width:10em
rem:
height: 10rem - width: 10rem
% (porcentagem):
height: 50% - width: 50%
ex:
heigth: 15ex - width: 15ex
ch:
height:17 - width: 17
