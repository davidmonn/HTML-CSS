CSS - Unidade de Medida Relativas (1/3)

As unidades de medidas relativas, iram mudar de tamanho baseado nos elementos que temos em nossa pagina.

Unidades Relativas:

  • Em: é baseada no font-size do elemento pai. Se o pai não tivesse um font-size definido, o filho herdaria do body (16px), ou seja: 1.5 * 16px = 24px.

  • REM (Root EM): É uma unidade de medida relativa ao tamanho da fonte definida no elemento raiz (html). Não importa o tamanho da fonte dos elementos pais, o rem sempre usa como base o valor do font-size do html. Por padrão, nos navegadores, o font-size do html é 16px (salvo se for alterado).


Nos exemplos abaixo, temos um box pai com um font-size de 20px.

em:
height: 10em - width:10em
rem:
height: 10rem - width: 10rem
