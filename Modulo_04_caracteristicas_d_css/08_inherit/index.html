CSS - Inherit

inherit significa herdar. É usado no CSS para que um elemento pegue o mesmo valor da propriedade do seu elemento pai. Ou seja, ele copia o valor que o pai está usando. Algumas propriedades já herdam automaticamente, como a cor do texto e a fonte. Outras, como margem e borda, não herdam, mas você pode forçar essa herança usando inherit. É útil quando você quer manter o estilo igual ao do pai sem precisar repetir o valor.

Basicamente o inherit serve para fazer um elemento herdar o valor de uma propriedade do elemento pai, mantendo o mesmo estilo.
