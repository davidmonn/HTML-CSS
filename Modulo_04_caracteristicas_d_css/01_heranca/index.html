CSS - Heranca

Aqui iremos ver um conceito muito importante do CSS que é herança.

herança: no CSS é quando um elemento HTML recebe automaticamente estilos (valores de propriedades) do seu elemento pai (ou ancestral), sem precisar ser declarado diretamente nele.

Exemplo:

Um bom exemplo de herança em CSS é quando temos uma < div > e, dentro dela, colocamos outros elementos, como textos.

Se no CSS selecionarmos essa < div > e aplicarmos color: red;, todo o texto dentro dela ficará vermelho.

Isso acontece porque o texto está herdando essa propriedade de cor da < div > onde está inserido.

É importante saber que não e todas as propriedades que sao herdadas, tem propriedades que não sao herdadas como border isso porque a propriedade border e uma propriedade que nao e herdada, normalmente as propriedades que sao herdadas sao referentes ao texto
