O option tem duas versoes de valores diferentes que podemos usar para ele, uma coisa e oque o usuario esta vendo, outra coisa e oque iremos enviar no formulario. Se formos na tag option e colocarmos o atributo value conseguimos definir um valor especifico para esse item que foi escolhido pelo usuario, visualmente para o usuario sera mostrado a opcao dentro da option e para os programadores pode receber com o nome usado no value. Quando nao colocamos um value, sera enviado com o mesmo nome do item exibido, ja se usarmos com value, sera enviado com o valor do value.
