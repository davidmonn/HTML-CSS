Label

Usamos o label, quando queremos basicamente definir um input como por exemplo: aqui sera usado para nome, outro para email, outro para senha e etc, visualmente para o usuario nao fara tanta diferenca, mas para a gente ficara basicamente assim:

sem label:
< input type="text" name="nome" >

Com label:
< label for="nome" >Nome: < / label >
< input type="text" name="nome" id="" >

No for de nosso label, usamos o name ou id de nosso input, para que ele possa ligar um ao outro. Apos ligado e muito util pois nao precisamos necessariamente clicar apenas na caixinha de nosso input mas tambem ao clicar em cima da palavra ligada com label, ira para nosso input.

Exemplo de uso:


Outra forma que temos de adicionar o label em nosso input, e colocar o input dentro da tag label.
Ex:

< label > Email
< input type="email" name="email" placeholder="Email:" >
< /label >

Essa segunda opcao vem sendo mais utilizada por dar um visual mais clean para o nosso codigo.
