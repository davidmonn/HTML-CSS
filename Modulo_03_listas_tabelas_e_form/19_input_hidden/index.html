Input type="hidden"

O input hidden e basicamente um campo que iremos enviar junto do formulario, mas o usuario nao precisa preencher. Muito util em alguns senarios, vamos supor que queremos uma informacao do usuario como local de acesso, onde nao queremos colocar como uma area de formulario para o usuario preencher, mas que pegue esse dado sem mostrar, com isso podemos usar o type hidden, onde nao sera mostrado para o usuario e assim que o formulario for enviado, sera enviado junto o hidden.

Geralmente esse campo e usado em capo para editar uma informacao do usuario, onde ira mostrar a conta, email e suas informacoes para editar, e quando formos enviar essas informacoes, junto da informacao, precisamos dizer qual o usuario estamos editando, entao junto do formulario provavelmente ira um campo hidden, identificando o usuario como um id.
