Tabelas no HTML:

Antigamente era comum os sites serem todos estruturados por meio de tabelas. Mas hoje em dia isso foi descontinuado, onde ela serve apenas para seu proposito original, sendo exibir dados de forma estruturada.

Para criar uma tabela iremos usar a tag table, apos feita, precisamos estrutura-la, ou seja, ver quantas linhas ela ira ter, colunas e etc.
