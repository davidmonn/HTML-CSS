Como usar CSS (2/2)

Temos 3 formas de incersao de nosso CSS em nosso HTML, sendo elas:

  • CSS Inline
  • CSS Interno
  • CSS Externo

Para usarmos o CSS Inline: iremos usar o atributo style e nesse atributo iremos adicionar os nossos codigos CSS, e tudo que colocarmos dentro desse atributo style, sera interpretado pelo navegador como CSS.
