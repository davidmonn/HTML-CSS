Width e Height

Nessa aula veremos as aplicacoes de largura (width) e altura (height).

widht: ira alterar a largura de um elemento.

height: ira alterar a altura de um elemento.

Temos muitas formas de escolher um valor de width (largura) e height (altura), podendo ser em px, cm, vh para height (viewport height), vw para largura (viewport width), %, inheret etc.

inherit: ele faz com que seu elemento com esse valor, herde a propriedade podendo ser altura ou largura do elemento pai.
