Base geral do CSS

Em nosso CSS temos formas diferentes de selecionarmos o mesmo elemento, um exemplo e, quando temos muitos elementos iguais em nossa pagina, todas herdaram o mesmo estilo, mas se quisermos que uma delas fique diferente, para isso temos 2 formas para pordemos fazer isso, chamados de: class e id.

Id: tem que ser unico para aquele elemento em especifico, ex se tivermos um elemento h1 com id de titulo, nao iremos poder colocar o mesmo nome de id em outro elemento, como um button com id de titulo, pois ele ja esta em uso. Para selecionarmos um id em nosso CSS iremos usar # mais o nome do id, ex: #titulo.

class: Mais utilizada, usada de forma parecida com id, porem a class permite que usemos os mesmo nomes em diferentes elementos. Para chamar uma class, usamos o . (ponto) mais o nome da class. Ex .texto

Outra forma de selecionarmos um elemento, e com o nome de sua tag.
